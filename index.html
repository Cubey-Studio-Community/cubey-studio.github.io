Cubey Studio

Cubey Studio is a complete IDE like Visual Studio 2026 and Xcode but this IDE is more compatible in Windows than MacOS but soon will support a owned OS that is CubeyOS!

For CubeyOS Apps (it will be soon), the recommended language is CubeyLang! For a code, use this code:

main.cby

require System

<System.outPrint>:print("Hello World!")

For a app, use this code:

main.cby

require CubeyStudio

pub class MyApp = [extends CubeyStudio.App] start
    fun Elements(var element) start
	subvar element.text = new CubeyStudio.Elements.Text("Hello World! This is my button!", "h1")
	subvar element.button = new CubeyStudio.Elements.Button("My Button!")
    end
end

Output/Window:

main.cby - Preview

Hello World! This is my button!

Features:
